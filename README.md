# 🔥 Youtube Cloud Comment 

## 설명
유튜브 비디오 위에 실시간으로 댓글을 오버레이하여, 동적인 시청 경험을 제공합니다.<br>
Google Cloud YouTube Data API를 통해 댓글을 불러옵니다.<br>

## 주요 기능
실시간 댓글 오버레이: 서버에서 댓글을 불러와 비디오 위에 표시합니다.<br>
반응형 디자인: 비디오 크기 변경에 따라 오버레이가 자동으로 조정됩니다.<br>
